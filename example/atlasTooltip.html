AtlasTooltip

v0.96

by Radney Aaron Alquiza

AtlasTooltip is a lightweight (8kb JS file + 6kb CSS file), easy-to-use jQuery plugin for use with any html element on the viewport.
This is still at beta stage so full functionality is still not completely implemented. There will be bugs.
Feel free to use it, and please report any bugs or features you want added to the plugin.

I've tested this against jQuery 1.7.1 and 1.9.1

Features as of v0.96:

  • Position tooltips: top, bottom, left, right.
  • Customizable colors (via the CSS file provided).
  • Pass any of the following: text, text containing html tags, DOM elements (jQuery, javascript objects).

Features coming to v1.0:

  • Customize delay of showing the tooltip. (hide is, for now, almost instant)
  • Control when to show tooltips, even without hovering over the element.

Bugs I've found so far:

  • No position awareness (yet) - the tooltip will be cut off if positioned near the edge of the viewport.
  • Multiple tooltips with the same position will overlap, one covering the other.
  • Positioning not fully centered relative to the element to be tipped.
